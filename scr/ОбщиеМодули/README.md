# Общие модули проекта

## Структура
- `РаботаСКэшем` - Модуль для работы с кэшем отчетов
- `Утилиты` - Общие утилитарные функции

## РаботаСКэшем
### Основные функции:
1. `ПолучитьИзКэша(Ключ, ВремяЖизниСекунды)` - получение данных из кэша
2. `СохранитьВКэш(Ключ, Данные)` - сохранение данных в кэш
3. `ИнвалидироватьКэшПриИзмененииПродаж(Дата)` - инвалидация кэша при изменении данных

### Особенности:
- Автоматическая очистка устаревших записей
- Защита от поврежденных данных
- Интеграция с журналом регистрации

## Утилиты
### Основные функции:
1. `ТаблицаВJSON(Таблица)` - преобразование таблицы в JSON
2. `ИзмеритьПроизводительностьЗапроса(Запрос)` - измерение времени выполнения
3. `СоздатьИндексыДляОптимизации()` - создание индексов для СУБД

## Расширения
### Назначение:
1. `РасширениеРеализации` - добавляет инвалидацию кэша при проведении документов
2. `РасширениеОтчетаТОПТоваров` - добавляет экспорт в Excel и дополнительные фильтры

### Принцип работы:
- Работают поверх существующих объектов
- Не изменяют исходный код конфигурации
- Легко отключаются при необходимости
